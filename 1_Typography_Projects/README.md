# Typography & Composition

This folder contains the main assembly scripts that use geometric primitives to construct complex typography.

## 🔠 Projects Included

### 1. UOA Construction (`main_UOA.m`)
* **U**: Built using two vertical lines and two concentric bottom semi-circles.
* **O**: Constructed with two concentric full circles (R=15, R=7).
* **A**: A fully polygonal construction using 8 distinct line segments for the body and 4 for the inner counter-space (the "hole").

### 2. GRAFIKA Rendering (`main_GRAFIKA.m`)
A complete word rendering showcasing:
* **Custom Curves:** The letter "Ρ" uses Kappel’s right-ellipse algorithm.
* **Complex Intersections:** The letter "Φ" combines a vertical axis with a centered circle.
* **Symmetry:** Consistent alignment and scaling across the entire string.
